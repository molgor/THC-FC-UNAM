1. Presentación del curso

  • Curso básico de herramientas computacionales para estudiantes de Ciencias
  • Vamos a formar habilidades técnicas fundamentales
  • Utilizaremos tecnologías libres y de código abierto

2. Temario

2.1. Módulo 1: Introducción a Linux/Unix

  • Anatomía de una computadora y un sistema operativo
  • Instalación y actualización de software en sistemas basados en Unix
  • Programas y comandos esenciales
  • Configuración del Shell
  • IDEs – guía básica de comandos en VIM y Tmux

2.2. Módulo 2: Introducción al análisis, modelación y visualización de datos con Python

  • Ecosistema de Python (paquetes, versiones, etc)
  • Manejo de Ambientes Virtuales
  • Sesiones interactivas
  • Instalación de paquetes
  • Tipos de datos en Python
  • Manejo de tablas con Pandas
  • Visualización de datos con Pandas
  • Análisis espacial con GeoPandas

2.3. Módulo 3: Métodos computacionales y visualización de datos

  • Computación simbólica
  • Simulación estocástica
  • Ajuste de modelos paramétricos
  • Introducción a la regresión lineal
  • Visualización de datos (correlogramas, scatterplots, boxplots, etc)

2.4. Módulo 4: Cómo hacer un reporte científico

  • Tex, Latex y otros lenguajes de Marcado para escribir documentos
  • El tríangulo de la Ciencia (Datos, Teoría y Modelos)
  • Reproducibilidad, Relevancia y Rigor
  • Estructura de un Artículo Científico
  • Bibtex y su integración con software para manejo de referencias
  • Presentación de Trabajos Finales

4.1. Forma de entrega de prácticas (Gitflow)

  1. Cada estudiante hace un fork del Repositorio Central
  2. Completa la práctica (Notebook interactivo)
  3. Hace una petición de integración (merge request) en una rama nueva con formato: /practica_n/Apellido_Nombre
  4. Antonio revisa la práctica y propone comentarios, habrá una oportunidad para corregir y volver a someter la práctica una sóla vez (esto contará como reposición). Las fechas de entrega y segundas vueltas se definirán durante el semestre.

5.2. Agregar llaves en Repositorio Github

Vamos a Github

5.2.1. Crear credenciales SSH

  1. Abre la terminal

  2. Ejecuta: 

    │  ssh-keygen -t ed25519 -C "tu_email@ejemplo.com"
  3. Cuando pregunte, elige ubicación para guardar las llaves (por defecto /.ssh/id_ed25519)
  4. Ingresa una contraseña de seguridad segura (opcional pero recomendado)

5.2.2. Archivos generados

  • /.ssh/id_ed25519 (llave privada - NUNCA com|partir)
  • /.ssh/id_ed25519.pub (llave pública - compartir con servicios)
